# TaskMaster - Flutter Task Management App

TaskMaster is a modern, responsive task management application built with Flutter. It helps users organize their tasks with features like due dates, task descriptions, and completion tracking.

## Features

- Create, edit, and delete tasks
- Mark tasks as complete/incomplete
- Set due dates for tasks
- Dark/Light theme support
- Responsive design that works on mobile and web
- Persistent storage using SharedPreferences
- Smooth animations and transitions

## Tech Stack

- **Framework**: Flutter
- **State Management**: Provider
- **Local Storage**: SharedPreferences
- **UI**: Material Design 3
- **Animations**: flutter_animate

## Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK (included with Flutter)
- Android Studio / VS Code with Flutter extensions
- Android Emulator or physical device for testing

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/codelyA23/Mobile-task-manager.git
   cd Mobile-task-manager
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## Project Structure

- `lib/` - Main application code
  - `models/` - Data models and providers
  - `screens/` - App screens
  - `theme/` - App theming
  - `widgets/` - Reusable widgets
  - `services/` - Business logic and services
